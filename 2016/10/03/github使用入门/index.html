github使用入门

介绍

github 是为开发者提供 GIT 仓库的托管服务。是一个让开发者与朋友、同事及陌生人共享代码的完美平台。

Github的作用

  • 托管仓库
  • 社交化编程（程序员的新浪微博）

基本使用

地址：https://www.github.com/

  1. 创建账户
    • 填写Username、Email Address、和Password
    • 如果有账户直接sign in
  2. 创建仓库
    • 登录完成后，右上角有一个加号，点击后出现下拉列表，选择 New repository
    • 填写仓库名称、描述、选择公开还是私有
    • 以上信息填写完成后点击 Create repository 创建，我们就有了一个远程仓库。

Git 与 Github 的区别

  • 在 GIT 中，开发者将原代码存入名叫“Git仓库”的资料库中并加以使用。
  • Github 为开发者的“Git仓库”提供了一个托管的平台，在该平台上公开的软件原代码全都由Git进行管理。
  • Github 除提供Git仓库的托管服务外，还为开发者提供了一系列功能，帮助高效率、高品质的进行代码编写。
  • Github创始人的愿望就是能有一个Git仓库托管服务让自己与朋友轻松分享代码。
