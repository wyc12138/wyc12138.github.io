JQuery入门(三)

JQuery 操作 DOM

JQuery 操作属性

  • 设置单个属性
    • attr(name,value);
    • 第一个参数：需要设置的属性名
    • 第二个参数：对应的属性值
    • 案例：$("img").attr("title","hello"); $("img").attr("alt","hello");

  • 设置多个属性

    • attr(obj)
    • 参数是一个对象，包含了需要设置的属性名和属性值
    • 案例：

  • 获取属性

    • attr(name)
    • 传需要获取的属性名称，返回对应的属性值

    • 案例：

    • 注意：获取属性值时，只会获取到第一个元素对应的属性，与 css 方法一样；如果该属性不存在。那么会返回 undefined

  • 移除属性
    • removeAttr(name)
    • 参数：需要移除的属性名，如果传空，那么不会有任何操作，注意，并不是移除所有的属性，区分removeClass
    • 用法举例：$("img").removeAttr("title")
  • prop
    • 设置属性：$(":checked").prop("checked",true);
    • 获取属性：$(":checked").prop("checked"); 返回 true 或者 false
    • 注意：在JQuery1.6之后，对于 checked、selected、disable 这类 boolean 类型的属性来说，如果使用 attr 方法获取属性值，得到的不是 true 和 false，而是 checked 以及 undefined。使用 prop 方法来获取或者设置 checked、selected、disable 这类的值。prop 方法使用跟 attr 方法一样。

JQuery 操作值与内容

  • val 方法
    • val 方法用于设置和获取表单元素的值，如：input、select、textarea的值
    • 设置值：$("#name").val("张三");
    • 获取值：$("#name").val();
  • html 方法
    • 设置内容：$("div").html("<span>这是一段内容</span>")
    • 获取内容：$("div").html()
  • text 方法
    • 设置内容：$("div").text("<span>这是一段内容</span>")
    • 获取内容：$("div").text()

html方法与text方法的区别：html方法会识别html标签，text方法会把内容直接当成字符串，并不会识别html标签

JQuery 操作尺寸

  • height 设置或获取高度
    • 带参数表示设置高度 $("img").height(200);
    • 不带参数获取高度 $("img").height();
  • width 设置或获取宽度，与高度操作功能一样
    • 带参数表示设置宽度 $("img").width(200);
    • 不带参数获取宽度 $("img").width();
    • 返回值是 number类型（如200），使用$(“img”).css(“width”)返回的是字符串（如200px）

JQuery 操作坐标值

  • offset
    • 设置或获取元素相对于文档 document 的位置
    • 设置位置：$(selector).offset({left:100,top:150});
    • 获取位置：$(selector).offset();
    • 注意：使用 offset 操作，如果元素没有设置定位（默认 position:static），则会把 position 修改为 relative 会修改 left、top
  • position
    • 获取相对于其最近的有定位的父元素的位置
    • 获取，返回值为对象：{left:num,top:num} $(selector).position();
    • 注意：position 方法只能获取，不能设置
  • scrollTop
    • 设置或者获取垂直滚动条的位置
    • 有参数表示设置偏移，参数为数值类型 $(selector).scrollTop(100)
    • 无参数表示获取偏移 $(selector).scrollTop()
  • scrollLeft
    • 设置或获取水平滚动条的位置
    • 有参数表示设置偏移，参数为数值类型 $(selector).scrollLeft(100)
    • 无参数表示获取偏移 $(selector).scrollLeft()

JQuery 事件机制

JQuery 对 JavaScript 事件进行了封装，增加并扩展了事件处理机制。JQuery 不仅提供了更加优雅的事件处理语法，而且极大的增强了事件的处理能力。

JQuery 事件的发展历程

简单事件绑定 >> bind 事件绑定 >> delegate 事件绑定 >> on 事件绑定（推荐）

  • 简单事件绑定
    • click（handler）单击事件
    • mouseenter（handler）鼠标进入事件
    • mouseleave（handler）鼠标离开事件
    • scroll（handler）滚动事件
    • 缺点：一次只能绑定一个事件
  • bind 事件绑定
    • 不推荐使用，JQuery1.7版本后被 on 取代
    • 绑定多个事件
      • 第一个参数：事件类型
      • 第二个参数：事件处理程序
      • $(“p”).bind(“clik mouseenter”,function(){
        事件响应方法
        })
    • 缺点：不支持动态创建出来的元素绑定事件
  • delegate 事件绑定
    • 支持动态事件绑定
      • 第一个参数：selector，要绑定事件的元素
      • 第二个参数：事件类型
      • 第三个参数：事件处理函数
        $(“.parentBox”).delegate(“p”,”click”,function(){
        为.parentBox 下面的所有的 p 标签绑定事件
        });
    • 支持原因：事件冒泡机制，事件是绑定到父元素上的，由子元素触发。
  • on 事件绑定
    JQuery1.7之后，JQuery用 on 统一了所有时间的处理方法

on 事件绑定

优势：最现代的方式，兼容zepto（移动端类似于JQuery的一个库），强烈建议使用。
第一个参数：events，绑定事件的名称可以是由空格分隔的多个事件（标准事件或自定义事件）
第二个参数：selector，执行事件的后代元素（可选），如果没有后代元素，那么事件将由自己执行。
第三个参数：date，传递给处理函数的数据，事件触发的时候通过event.data来使用（不常使用）
第四个参数：handler，事件处理函数 $(selector).on(events[selector],[data],handler);
例子：

  • 表示给$(selector)绑定代理事件，当必须是它的内部元素 span 才能触发这个事件，支持动态绑定
    • $(selector).on(“click”,”span”,function(){});
  • 表示给$(selector)绑定事件，并且由自己触发，不支持动态绑定（不使用代理）
    • $(selector).on(“click”,function(){});

事件解绑

  • unbind() 方式（不用）
    • 作用：解绑事件
      • 解绑所有事件 $(selector).unbind();
      • 解绑指定事件 $(selector).unbind("click");
  • undelegate() 方式（不用）
    • 作用：解绑事件
      • 解绑所有的 delegate 事件 $(selector).undelegate();
      • 解绑所有的 click 事件 $(selector).undelegate("click");
  • off方式
    • 作用：解绑事件
      • 解绑匹配元素的所有事件 $(selector).off();
      • 解绑匹配元素的所有 click 事件 $(selector).off("click");
      • 解绑匹配所有代理的 click 事件，元素本身的事件不会被解绑 $(selector).off("click","**");

事件触发

  • 简单事件触发
    • $(selector).click(); 触发 click 事件
  • trigger 方法触发事件
    • $(selector).trigger("click");
  • triggerHandler 触发 事件响应方法，不触发浏览器行为
    • $(selector).triggerHandler("focus");
